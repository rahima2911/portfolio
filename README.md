# Rahima — Portfolio

A personal portfolio site built with React, Vite, and Tailwind CSS — with a working chatbot that answers visitor questions about the skills, experience, and projects shown on the site.

## Tech stack

- React + Vite
- Tailwind CSS, with a light/dark theme toggle
- A small client-side chatbot (TF-IDF + cosine similarity matching — no API key or backend needed)

## Running locally

```bash
npm install
npm run dev
```
